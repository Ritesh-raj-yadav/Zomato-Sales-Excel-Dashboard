# 📊 Zomato Sales Data Analysis Dashboard (Excel)

## Overview

Built an interactive Excel dashboard to analyze Zomato sales data, focusing on performance tracking, trend analysis, and business insights. The project demonstrates practical data analysis, data cleaning, and visualization skills.

## Problem Statement

Raw sales data is unstructured and hard to interpret. The goal was to transform it into a structured, insight-driven dashboard for quick decision-making.

## Tools & Skills

* Microsoft Excel 
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Data Analysis & KPI Tracking
* Dashboard Design

## Data Preparation

* Removed duplicates and handled missing values
* Standardized inconsistent data formats
* Used Excel functions for cleaning
* Created calculated columns for metrics

## Dashboard Features

* Dynamic and interactive dashboard using slicers
* KPI metrics: Total Sales, Total Orders, Average Rating
* City-wise and cuisine-wise performance analysis
* Trend visualization using charts

## Key Insights

* High sales concentration in specific cities
* Certain cuisines consistently outperform others
* Repeat ordering patterns observed among customers
* Ratings show a direct impact on sales performance

## Impact

* Converts raw data into actionable insights
* Helps identify growth opportunities and top performers
* Demonstrates real-world business analysis skills using Excel

## How to Run

1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers to filter and explore insights
